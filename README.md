# dotfiles

## Usage

1. install arch linux: [guide](./install.sh)
2. `pacman -S git && git clone --depth 1 https://github.com/cybernty/dotfiles.git && cd dotfiles`
3. `./configure.sh`

## List

| type                 | use             |
| -------------------- | --------------- |
| window manager       | dwm             |
| shell                | zsh             |
| terminal             | alacritty       |
| terminal multiplexer | tmux, zellij    |
| editor               | neovim + vscode |
| file manager         | yazi            |
| browser              | chromium        |
| package manager      | yay             |
